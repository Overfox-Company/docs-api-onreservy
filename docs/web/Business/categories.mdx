# Get Categories

## Endpoint

```ts
GET https://www.onreservy.com/api/web/categories
```

## Use
In order to get the categories list, you need to provide the session token through the request header like this:
```ts
Authorization: Bearer token
```

### <span style={{color: "rgb(255, 39, 39)"}}>Failure</span>

In the event that a token is not sent or is invalid, the API will return a 401 status code with the following message:

```json
{
     "message": "Unauthorized"
}
```
